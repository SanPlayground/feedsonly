# RSS Reader

A modern RSS feed reader built with React, TypeScript, and Tailwind CSS.

## Features

- Clean, responsive UI
- Real-time feed updates
- Date formatting
- External link handling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Technology Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- React Query
- date-fns
- Lucide Icons
